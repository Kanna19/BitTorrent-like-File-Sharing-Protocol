Compile tracker using ```g++ -std=c++11 -pthread tracker.cpp -o tracker```

Run tracker using ```./tracker <Port Number>```

Compile client using ```g++ -std=c++11 -pthread client.cpp -o client```

Run client using ```./client <Torrent filename>```
